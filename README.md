#  Movie Review Sentiment Analysis

An end-to-end NLP project that classifies IMDB movie reviews into positive and negative sentiment.

##  Key Results
- Best Model: Logistic Regression (89.46%)
- TF-IDF outperformed BoW by +2.62%
- Strong generalization (no overfitting)
